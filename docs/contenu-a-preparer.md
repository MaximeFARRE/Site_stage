# Contenu à préparer avant de coder

## Identité

- nom complet ;
- phrase d’accroche ;
- bio courte ;
- bio longue ;
- photo professionnelle.

---

## Liens

- LinkedIn ;
- GitHub ;
- email ;
- CV PDF.

---

## Projets à préparer

Minimum 3 projets très solides.

Exemple :
- application de suivi de patrimoine ;
- projet de backtest / portefeuille ;
- Occifloc.

Pour chaque projet, préparer :
- titre ;
- résumé ;
- problème ;
- solution ;
- résultats ;
- captures d’écran ;
- outils utilisés.

---

## Expériences

Pour chaque expérience :
- titre ;
- dates ;
- rôle ;
- missions ;
- résultats.

---

## Compétences

Classer toutes les compétences en 3 catégories :
- finance ;
- technique ;
- business.
