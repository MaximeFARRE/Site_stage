# Design System

## Objectif visuel

Le site doit transmettre :
- professionnalisme ;
- clarté ;
- ambition ;
- sobriété.

---

## Style général

- design minimaliste ;
- beaucoup d’espace ;
- cartes propres ;
- peu de couleurs ;
- animations discrètes.

---

## Palette recommandée

- Fond principal : blanc cassé ou très clair
- Texte principal : gris très foncé
- Couleur accent : bleu profond ou bleu nuit
- Couleur secondaire : gris bleuté léger

Exemple :
- `#F8F9FB`
- `#111827`
- `#1D4ED8`
- `#64748B`

---

## Typographies

- Titres : Inter, Geist ou Sora
- Texte : Inter

---

## Règles d’espacement

- grandes marges entre les sections ;
- largeur maximale du contenu : 1100 à 1200 px ;
- ne jamais coller les éléments.

---

## Règles UI

### Boutons

Deux variantes uniquement :
- bouton principal ;
- bouton secondaire.

### Cartes

Toutes les cartes doivent avoir :
- mêmes coins arrondis ;
- même ombre ;
- même espacement interne.

### Icônes

Toujours utiliser une seule bibliothèque.

Recommandation : Lucide.
