## Projets

Objectif : montrer les réalisations les plus fortes.

Chaque carte projet doit contenir :
- image ;
- titre ;
- courte description ;
- technologies ;
- lien vers la fiche détaillée.

---

## Projet détaillé

Chaque projet détaillé doit suivre exactement cette structure :

1. Contexte
2. Objectif
3. Problème
4. Solution
5. Outils utilisés
6. Résultat
7. Ce que j’ai appris
